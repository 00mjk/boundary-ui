# Footer

A footer component that allows four distinct sections to be configured: brand, nav, product, and copyright.

```html
<Rose::Footer as |footer|>
  <footer.brand @logo="logo" @text="Product Name" />
  <footer.copyright @text="Company Name" />
  <footer.product @text="Product Name" @version="1.0" />
  <footer.nav as |nav|>
    <nav.external-link @route="index">Section</nav.external-link>
    <nav.external-link @route="about">Section</nav.external-link>
  </footer.nav>
</Rose::Footer>
```
