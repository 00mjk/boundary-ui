# Tabs

A tabs layout component to hold `Rose::Tab` definitions.

```html
<Rose::Layout::Tabs>
  <Rose::Tab @title="Tab item" @badge="3" @icon="info-circle-outline" @disabled={{false}} />
</Rose::Layout::Tabs>
```
