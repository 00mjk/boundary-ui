# Layout|Global

A top-level layout/positioning component that allows three distinct sections to be configured: header, navigation, and body.

```htmlbars
<Rose::Layout::Global as |global|>
  <global.header />
  <global.body />
  <global.navigation />
</Rose::Layout::Global>
```
