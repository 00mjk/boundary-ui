# Layout|Page

A page layout/positioning component that allows three distinct sections to be configured: header, sidebar, and main.
Sidebar's content width determines it's column width.

```htmlbars
<Rose::Layout::Page as |page|>
  <page.header />
  <page.sidebar />
  <page.main />
</Rose::Layout::Page>
```
