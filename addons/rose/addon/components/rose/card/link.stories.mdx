# Card Link

Card link component wraps content with `LinkTo` component. Supports title and description.

```html
<Rose::Card::Link @title='Card title' @description='Card description' @route='about'/>
```

<Preview>
  <Story name="Basic">{{
    template: hbs`
      <Rose::Card::Link
        @title={{title}}
        @description={{description}}
      />
    `,
    context: {
      title: text('title', 'Card title'),
      description: text('description', 'Card description'),
    },
  }}</Story>
</Preview>

`@route` defaults to 'index' when not supplied.

`@title` and `@description` are optional. Card expands to fit content.

```html
<Rose::Card::Link @title='Card title'/>
```
<Preview>
