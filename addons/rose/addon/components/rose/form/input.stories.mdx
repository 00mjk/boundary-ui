# Form Input

Renders a simple form input.

```htmlbars
<Rose::Form::Input
  @type="email"
  @name="email"
  @value="info@example.com"
  @label="Email"
  @helperText="Enter your email"
  @error={{false}}
  @disabled={{false}}
  readonly={{false}}
/>
```
