# Form Textarea

Renders a simple form textarea.

```htmlbars
<Rose::Form::Textarea
  @value="Descriptive content"
  @label="Description"
  @helperText="Enter a description"
  @error={{false}}
  @disabled={{false}}
  readonly={{false}}
/>
```
