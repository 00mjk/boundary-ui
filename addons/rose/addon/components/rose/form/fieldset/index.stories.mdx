# Form Fieldset

Renders a simple fieldset with title, description and body.

```html
<Rose::Form::Fieldset>
  <:title>Title</:title>
  <:description>Description</:description>
  <:body>Rest of the form</:body>
</Rose::Form::Fieldset>
```
