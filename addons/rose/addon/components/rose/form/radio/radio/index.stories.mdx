# Form Radiofield

Renders a form radiofield.

```htmlbars
<Rose::Form::Radio::Radio
  @label="Yes"
  @error={{false}}
  @value="yes-agree"
  @selectedValue="yes-agree"
  @disabled={{false}}
  />
```
