# Title

A header title component with ability to configure dropdown using `Rose::Header::Dropdown`. To be used with `Rose::Layout::PageHeader`.

```html
<Rose::Layout::PageHeader style="border: 1px solid var(--brand-gray-5);">
  <Rose::PageHeader::Title as |title| >
    <title.dropdown
      @text="Page Header Title"
      @icon="user-square-fill" as |dropdown|
    >
      <dropdown.link @route="index">Menu item</dropdown.link>
      <dropdown.button>Button menu item</dropdown.button>
    </title.dropdown>
  </Rose::PageHeader::Title>
</Rose::Layout::PageHeader>
```

Title can be configured to be a level 1 heading with optional icon.

```html
<Rose::PageHeader::Title @text="Page Header Title" @icon="alert-circle-fill" />
```
