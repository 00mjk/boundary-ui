# Utilities

An component comprised of `Rose::Button` as actions. To be used with `Rose::Layout::PageHeader`.

```html
<Rose::PageHeader::Utilities as |utilities|>
  <utilities.action @style="secondary">Secondary</utilities.action>
  <utilities.action @style="primary">Primary</utilities.action>
</Rose::PageHeader::Utilities>
```
